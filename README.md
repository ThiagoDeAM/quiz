# Quiz App (React Native + SQLite)

Aplicativo desenvolvido com **React Native** usando Expo, que implementa um **jogo de perguntas e respostas**.  

O jogo seleciona **10 perguntas aleatórias** do banco, apresenta duas opções de resposta para cada pergunta e, ao final, mostra ao usuário quantas perguntas foram acertadas.

---

## Funcionalidades

- Botão **Iniciar** para começar o jogo
- Sorteio de **10 perguntas aleatórias**
- Exibição das perguntas
- Avanço automático para a próxima pergunta após resposta
- Exibição do resultado final com número de acertos
- Opção para **reiniciar o jogo**

---

## Como executar o projeto

```bash
npx expo start
```
